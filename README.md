# Autonomous Job Agent

An end-to-end AI system that:
- Discovers jobs from real company job boards
- Ranks roles using semantic + lexical signals
- Enriches full job descriptions
- Parses candidate resumes
- Generates tailored, professional cover letters using a local LLM
